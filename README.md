# Vehicle-Parking-Management-System
🚗 Vehicle Parking Management System The Vehicle Parking Management System is a web-based application designed to simplify and automate four-wheeler parking operations.

🔑 Key Features
Admin Login: Admins can create and manage parking lots, monitor spot availability, and oversee reservations.

User Login: Users can register, log in, and book available parking spots in real time.

Multiple Parking Lots & Spots: Scalable design to handle several parking areas simultaneously.

Reservation Management: Track parking time, leaving time, and calculate costs automatically.

Responsive UI: Built with Bootstrap for a clean and mobile-friendly interface.

🛠️ Tech Stack
Frontend: HTML, CSS, Bootstrap

Backend: Python (Flask framework)

Database: SQLite (SQLdbLite)

Version Control: Git & GitHub

⚙️ How to Run the Project
Follow these steps to set up and run the Vehicle Parking Management System locally:

1. Clone the Repository
bash
git clone https://github.com/ChahatPawarCS/Vehicle-Parking-Management-System.git
cd Vehicle-Parking-Management-System
2. Create a Virtual Environment
bash
python -m venv venv
Activate it:

Windows (PowerShell):

bash
venv\Scripts\activate
Linux/Mac:

bash
source venv/bin/activate
3. Install Dependencies
bash
pip install -r requirements.txt
(Make sure you have a requirements.txt file listing Flask, Bootstrap, and other dependencies.)

4. Run the Application
bash
python app.py
5. Access in Browser
Open your browser and go to:

Code
http://127.0.0.1:5000/
🧑‍💻 Default Roles
Admin Login: Manage parking lots, spots, and users.

User Login: Book and release parking spots in real time.
